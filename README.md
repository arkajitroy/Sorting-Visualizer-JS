
# Sorting Visualizer :bar_chart:

This is a very basic sorting visualizer project which is made using **Vanilla JavaScript**. It consist of all the basic sorting algorithms.

**Sorting Algorithms**
1. Bubble Sort
2. Insertion Sort
3. Selection Sort
4. Merge Sort
5. Quick Sort
